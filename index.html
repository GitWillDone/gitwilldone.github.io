TCG Images Repository

About

This is a GitHub Pages site hosting Trading Card Game (TCG) images. The URLs provided here can be used in backend projects to generate CSV files for eBay's bulk upload tool.

Base URL

All images are accessible via the base URL:

https://gitwilldone.github.io/tcg-images.github.io/images/

How to Use

Image URL Format:

https://gitwilldone.github.io/tcg-images.github.io/images/{set-name}/{card-name}.jpg

Example:

https://gitwilldone.github.io/tcg-images.github.io/images/base-set/charizard.jpg

Directory Structure

images/
├── base-set/
│   └── (card images)
├── fossil/
│   └── (card images)
├── jungle/
│   └── (card images)
└── (other sets...)

Image Manifest

A JSON manifest of all available images is available at:

https://gitwilldone.github.io/tcg-images.github.io/manifest.json

This manifest can be used to programmatically access all image URLs for bulk operations.
